# La Barbería — Sitio web

Landing page estática para barbería: hero, servicios, galería, testimonios y contacto por WhatsApp.

## Cómo ver el sitio

Abre en el navegador:

- `barberia/index.html`

O desde la raíz del proyecto, `barberia.html` redirige a la carpeta `barberia/`.

## Estructura

```
barberia/
├── index.html
├── css/
├── js/
├── partials/
└── assets/images/galeria/   ← fotos JPG de la galería
```

## Personalizar

- **Galería:** reemplaza los `.jpg` en `barberia/assets/images/galeria/`
- **WhatsApp / teléfono:** busca `526671234567` en `barberia/index.html`
- **Estilos:** archivos en `barberia/css/`

## Tecnologías

HTML, CSS y JavaScript vanilla (sin build ni dependencias).
